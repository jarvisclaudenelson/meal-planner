# Family Meal Planner

A mobile-first PWA for planning weekly family dinners and lunches, building a shopping list, and cooking with step-by-step guidance.

**Stack:** React 18 · Vite · Tailwind CSS · Supabase · vite-plugin-pwa

---

## Quick Start

### 1. Create a Supabase project

1. Go to [supabase.com](https://supabase.com) and create a free project.
2. In the SQL editor, run the contents of **`supabase/schema.sql`** to create the tables.
3. Copy your **Project URL** and **anon public key** from *Project Settings → API*.

### 2. Configure environment variables

```bash
cp .env.example .env
```

Edit `.env`:

```
VITE_SUPABASE_URL=https://your-project-ref.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key-here
```

### 3. Install dependencies & generate icons

```bash
npm install
node scripts/generate-icons.js   # creates public/pwa-192x192.png and public/pwa-512x512.png
```

### 4. Seed starter recipes (optional)

```bash
npm run seed
```

This inserts 5 recipes (Beef Burgers, Sheet Pan Chicken Thighs, Turkey Taco Bowls, Slow Cooker Chili, Herb-Crusted Salmon).

### 5. Run locally

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser (or add it to your home screen for the full PWA experience).

## Features

| Tab | What it does |
|-----|-------------|
| **Today** | Shows today's dinner & lunch. Calculates "start cooking by" time based on your target dinner time. |
| **Plan** | Weekly view (Mon–Sun). Tap a slot to assign a recipe; tap a filled slot to view it; clear with ✕. Navigate weeks with arrows. |
| **Recipes** | Browse, search, filter (Starred / Quick / High Protein / Kid-Friendly), and sort. Tap to view full recipe. |
| **Recipe Detail** | Full ingredients + steps + nutrition. Cooking Mode shows one step at a time with wake lock. Low-protein recipes show protein-boost tips. |
| **Shopping** | Auto-generate from the week's plan. Items grouped by store section. Check off items, add custom items, export as plain text to clipboard. |

## Database Schema

| Table | Key columns |
|-------|-------------|
| `recipes` | `id`, `name`, `servings`, `prep_time_min`, `cook_time_min`, `total_time_min` *(generated)*, `protein_g`, `calories`, `protein_ratio` *(generated: g/100 cal)*, `ingredients` (jsonb), `steps` (text[]), `tags` (text[]), `starred` |
| `meal_plans` | `week_start` (date), `day`, `slot` (dinner/lunch), `recipe_id` — unique per week+day+slot |
| `shopping_lists` | `week_start` (date, unique), `items` (jsonb array) |

```bash

```

2. Import the project in [vercel.com](https://vercel.com).
3. Add environment variables:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
4. Deploy — Vercel auto-detects Vite.

## Adding Recipes

You can add recipes directly in the Supabase table editor or write additional seed scripts following the pattern in `scripts/seed.js`.

Each ingredient should follow the shape:
```json
{ "item": "Chicken breast", "qty": 1.5, "unit": "lb", "section": "Meat & Seafood" }
```

Valid sections: `Produce`, `Meat & Seafood`, `Dairy & Eggs`, `Pantry`, `Frozen`, `Bread`, `Other`
